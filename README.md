# Api-Taller-Bicicletas
Sistema de gestión para taller de bicicletas desarrollado con Java y Spring Boot. Permite registrar bicicletas, crear órdenes de trabajo, asignar servicios, gestionar estados y controlar la capacidad del taller. Incluye arquitectura en capas (controller, service, repository) y base de datos MySQL.
